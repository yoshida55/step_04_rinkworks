# コーディング規約

## 命名規則

- 単語の区切りはアンダースコア（`top_title`, `logo_image`）
- ハイフン不可（`section-wrap` ❌）
- タグ名を入れない（`main_p` ❌ → `main_text` ⭕）
- ローマ字不可（`oshirase` ❌ → `news` ⭕）
- 極端な省略不可（`im` ❌ → `image` ⭕）

## フォーマット

- インデントはスペース 2 個
- コードは全て小文字
- 長いテキストはコード内で改行せず、VS Code の折り返し表示を使う

## 外部リンク

```html
<a href="https://example.com" target="_blank" rel="noopener noreferrer"></a>
```

## 画像の alt

文章ではなく名詞で書く（`これは自転車です` ❌ → `自転車` ⭕）

## セレクタ

- 深掘りは 1 段階まで
- タグを直接指定しない
- `position: absolute` は最終手段

## 【詳細解説】わかりにくいルール 3 つ

### 1. セクションの役割はコメントで残す

HTML を後から見たとき「ここ何のセクションだっけ？」とならないように、
コメントで目印を残す。

```html
<!-- ❌ NG：コメントなし。後から見ると何が何だかわからない -->
<section>
  <h1>Recipe Diary</h1>
  <p>日々の料理レシピをまとめています。</p>
</section>
<section>
  <img src="curry.jpg" alt="カレー" />
</section>
<footer>
  <p>©2021 Recipe Diary</p>
</footer>
```

```html
<!-- ⭕ OK：セクションごとに役割を書いておく -->

<!-- ==============================
  サイト紹介
============================== -->
<section class="intro">
  <h1>Recipe Diary</h1>
  <p>日々の料理レシピをまとめています。</p>
</section>

<!-- ==============================
  レシピギャラリー
============================== -->
<section class="gallery">
  <img src="curry.jpg" alt="カレー" />
</section>

<!-- ==============================
  フッター
============================== -->
<footer class="footer">
  <p>©2021 Recipe Diary</p>
</footer>
```

**なぜ？**
→ 100 行、200 行と増えたとき、コメントがないとスクロールして探し回ることになる。

---

### 2. リスト・テーブルは改行してインデントする

入れ子構造は「親子関係」が見えるように書く。

```html
<!-- ❌ NG：1行に詰め込んでいる -->
<ul>
  <li>Instagram</li>
  <li>Twitter</li>
  <li>Facebook</li>
</ul>
```

↑ これだと「li が ul の中にある」という構造が見えにくい。

```html
<!-- ⭕ OK：改行とインデントで親子関係が見える -->
<ul>
  <li>Instagram</li>
  <li>Twitter</li>
  <li>Facebook</li>
</ul>
```

**なぜ？**
→ 後から「li を 1 個追加したい」となったとき、どこに書けばいいか一目でわかる。

---

### 3. 一連の文章を 1 行ずつ p タグで区切らない

「見た目で改行したいから」という理由で p タグを分けない。

```html
<!-- ❌ NG：見た目の改行のためにpタグを分けている -->
<p>日々の料理レシピをまとめています。</p>
<p>和食や洋食、中華、お菓子まで</p>
<p>いろいろな料理レシピをアップしています。</p>
```

↑ これだと p タグ間に余計な余白ができる。しかも意味的に「3 つの段落」になってしまう。

```html
<!-- ⭕ OK：ひとつの段落はひとつのpタグ -->
<p>
  日々の料理レシピをまとめています。 和食や洋食、中華、お菓子まで
  いろいろな料理レシピをアップしています。
</p>
```

行間を空けたい場合は CSS で調整する：

```css
p {
  line-height: 1.8; /* 行の高さ */
}
```

**なぜ？**
→ p タグは「段落（意味のまとまり）」を表す。
→ 見た目のために HTML をいじると、後でデザイン変更したとき全部直す羽目になる。
